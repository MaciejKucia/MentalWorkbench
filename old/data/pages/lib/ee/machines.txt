==== Open circuit ====
  * All power --> loses (Cu, Fe)
  * $P_{10} \approx \Delta P_{Fe}$
  * $I_0 = I_\mu = \frac{E_1}{X_\mu} = \frac{U_1}{X_\mu}$
  * turns ratio: $n=\frac{U_{1N}}{U_{20}}=\frac{U_{1N}}{U_{2N}}$
  * core loses: $\Delta P_{Fe} = P_o - (R_1 I_0^2) \approx P_0$
  * power factor: $\cos \varphi_0 = \frac{P_0}{U_{1N}I_0}$
  * $I_{Fe}=\frac{\Delta P_{Fe}}{E_1} \approx \frac{P_0}{U_{1N}}$
  * $I_\mu = \sqrt{I_0^2 - I^2_{Fe}}$
  * $R_{Fe} = \frac{E_1^2}{\Delta P_{Fe}} \approx \frac{U^2_{1N}}{P_0}$
  * $X_\mu = \frac{E_1}{I_\mu} \approx \frac{U_{1N}}{P_0}$


==== Short circuit ====

  * supply voltage required to circulate rated current: $u_z=\frac{U_z}{U_N}$
  * $I_0 = I_/mu + I_{Fe} \approx 0$
  * $R_z = R_1 + R'_2$
  * $X_z = X_1 + X'_2$
  * $Z_z = \sqrt{R_z^2 + X_z^2}$
  * $U_z = Z_z I_N$
  * $U_N = Z_z I_z $
  * $\frac{U_z}{U_N} = \frac{I_N}{I_z}$
  * $\Delta P_{Cu} \approx P_z$
  * $\cos \varphi_z = \frac{P_z}{U_z I_N}$
  * $Z_z = \frac{U_z}{I_N}$
  * $R_z = \frac{\Delta P_{Cu}}{I^2_N} \approx \frac{P_z}{I^2_N}$
  * $R_z = R_1 + R'_2$ and $R_1 \approx R'_2$
  * $X_Z = \sqrt{Z_z^2 - R_z^2}$
  * $X_Z = X_1 + X'_2$ where $X_1 \approx X'_2$

  * $Z_z = \frac{U_z}{I_N}$
  * $u_{z\%} = \frac{U_z}{U_N} 100\% $
  * $S_N = U_N I_N$
  * **$Z_z = \frac{u_z}{100\%} \frac{U_N^2}{I_N}$**
  * **$R_z = \frac{\Delta P_{Cu}}{I^2_N} = \Delta P_{Cu} \frac{U_N^2}{S_N^2}$**

==== 3 phase ====

  * all powers x3 (phase values)
  * $Y$ $U=\sqrt{3} U_f$, $I=I_f$
  * $\triangle$ $U=U_f$, $I=\sqrt{3} I_f$
===== AC motor =====
==== Induction (asynchronous) motor ====

  * Synchronous speed: $n_1 = \frac{60f_1}{p}$
  * Field speed: $n_2 = n_1 - n$
  * Slip: $s = \frac{n_1 - n}{n_1}$ (0 when stall 1 when nominal)
  * $P_{in} = m_1 U_{1f} I_1 \cos \varphi_1$
  * $\Delta P_{Cu1} = m_1 R_1 I_1^2$
  * $\Delta P_{Fe1} = m_1 R_{Fe} I_{Fe}^2 = m_1 E_1 I_{Fe} = m_1 E'_{20}I_{Fe}$
  * $P_\Psi = P_{in} - \Delta P_{Cu1} - \Delta P_{Fe1} = m_2 \frac{R_2 + R_d}{s}I_2^2 = m_1 \frac{R'_2 + R'_d}{s} I'^2_2$
  * or: $P_\Psi =m_2 E_{20}I_2 \cos \varphi_2 = m_1 E'_{20} I'_2 \cos \varphi_2$
  * $\Delta P_{Cu2} = m_2 R_2 I^2_2 = m_1 R'_2 I'^2_2$
  * $P_2 = m_2 R_d I_2^2 = m_1 R'_d I_2^{'2}$
  * $P_{el} = \Delta P_{Cu2} + P_2 = m_2 \left( R_2 + R_d\right) I_2^2 = m_1 (R'_2 + R'_d) I'^2_2$
  * mechanical: $P_m = P_\Psi - P_{el} = m_2 \frac{1-s}{s}(R_2 + R_d) I^2_2 = m_1 \frac{1-s}{s}(R'_2 +R'_d) I'^2_2$
  * or: $P_m = \omega M = 2 \pi nM$
  * $P = P_m - \Delta P_m$
  * $P_\Psi = P_{el} + P_m$
  * $P_{el} = sP_\Psi$ $P_m = (1-s)  P_\Psi$
  * $\eta = \frac{P}{P_{in}} = \frac{P_{in} - \Sigma \Delta P}{P_{in}}$

  * $M = \frac{P_\Psi}{\omega_1} \approx 9,55 \frac{P_\Psi}{n_1} \approx 9,55 \frac{P_m}{n}$
  * $$M = \frac{m R'_2 U'^2_1}{2 \pi n_1 \left[ s(X_1 + X'_2)^2 + \frac{R'^2_2}{s} \right]}$$
  * $s_k = \pm \frac{R'_2}{X_1 + X'_2}$
  * $\frac{M_k}{M_N} = u$

=== No load ===
  * $\Delta P_{Cu0} = m_1 R_1 I_0^2$
  * $\Delta P_{Fe0} = m_1 \frac{U_1^2}{R_{Fe}}$
  * $P_0 = \Delta P_{Cu0} + \Delta P_{Fe0} + \Delta P_m $

=== Short ===
  * $P_z = \Delta P_{Cu1} + \Delta P_{Cu_2} $
  * $U_{z\%} = \frac{U_z}{U_N} 100\%$
  * $I_z = I_N \frac{100}{U_{z\%}}$
=== Load ===
  * start $$ I_r = \frac{U_{1f}}{\sqrt{(R_1 + R'_2)^2 + (X_1 + X'_2)^2}} $$
  * $$I_N \approx \frac{U_{1f}}{\sqrt{ \left( R_1 + \frac{R'_2}{s_N} \right)^2 + (X_1 + X'_2)^2 }}$$

=== Nominal params ===
  * $P_{1N} = \sqrt{3} U_{1N} I_{1N} \cos \varphi_{1N}$
  * $\eta_N = \frac{P_N}{P_{1N}}$ and $\Delta P_N = \Delta P_{1N} - P_N$
  * $M_N = \frac{P_N}{\omega_N} = \frac{P_N}{2 \pi n_N}$
  * $p = \frac{f_{1N}}{n_1}$ (n1 - synch)
  * $s_N = \frac{n_1 - n_N}{n_1}$
  * $\Delta P_{Cu2N} = P_{el} \approx \frac{s_N}{1 - s_N} P_N $
  * $n_u = n_i = \frac{E_{1N}}{E_{20N}} = \frac{U_{1Nf}}{U_{2Nf}}$
  * $R_2 = \frac{P_{el}}{3I^2_{2Nf}} = \frac{ \omega_{1N} - \omega_1}{\omega_N} \frac{P_N}{3 I^2_{2Nf}}$



==== Synchronous motor ====

  * $f = \frac{p n}{60}$
  * $P = mUI \cos \varphi$ (phase)
  * $M = 9,55 \frac{P}{n_1}$
